# Sample Data Directory

Place your fencing video files here for E2E testing.

## Recommended Video Specifications

| Type | Resolution | FPS | Duration |
|------|------------|-----|----------|
| Clean bout | 1920x1080 | 30 | 2-5 min |
| Infighting | 1920x1080 | 30 | 30-60 sec |
| Club noise | 1920x1080 | 30 | 2-3 min |

## File Naming

- `clean_bout.mp4` - Clean bout with clear touches
- `infighting.mp4` - Close-range fighting
- `club_noise.mp4` - With audience/referee noise

## Note

Video files are excluded from git via `.gitignore`.
